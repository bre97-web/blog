MVVM & Vue

Vue不是严格的MVVM模型的框架.

MVVM模型包括三个部分:

  • Model
  • ViewModel
  • View

Model

Model负责管理数据. Vue中的data, computed, methods, setup可以视为Model部分, 在.vue文件中的<script></script>标签也可以视为Model部分.

const componentA = defineComponent({
  setup() {
    const name = ref("I am component A");
    return { name };
  },
  data: () => ({
    message: "Hello Vue!",
  }),
  computed: {
    reversedMessage() {
      return this.message.split("").reverse().join("");
    },
  },
  methods: {
    reverseMessage() {
      this.message = this.reversedMessage;
    },
  },
});

<script setup lang="ts">
const message = ref("Hello Vue!");
const reversedMessage = computed(() => message.value.split("").reverse().join(""));
const reverseMessage = () => {
  message.value = reversedMessage.value;
};
</script>

View

View负责将Model中的数据渲染成可读的图形界面. Vue中的template和render可以视为View部分.

  • template提供声明式的视图编写方式
  • render提供可编程的视图编写方式
<template>
  <div>
    <p>{{ message }}</p>
  </div>
</template>

const componentA = defineComponent({
  data: () => ({
    message: "Hello Vue!",
  }),
  template: `
        <div>
          <p>{{ message }}</p>
        </div>
    `,
});

const componentA = defineComponent({
  data: () => ({
    message: "Hello Vue!",
  }),
  render() {
    return h("div", [h("p", this.message)]);
  },
});

const componentA = defineComponent({
  data: () => ({
    message: "Hello Vue!",
  }),
  render() {
    return (
      <div>
        <p>{this.message}</p>
      </div>
    );
  },
});

ViewModel

ViewModel负责链接Model和View. 在Vue中, vue应用程序实例可以视为ViewModel角色, vue实例在介入网页后追踪Model层的变化, 并将Model层的变化反映到View层, 同时监听View层的交互, 并将View层的交互反馈到Model层.

const appInstance = createApp({
  // ...
});

appInstance.mount("#root");
